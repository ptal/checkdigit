Detailed Description

template<unsigned int number_of_virtual_value_skipped = 0>
class boost::checks::visa_algorithm< number_of_virtual_value_skipped >

This class can be used to compute or validate checksum with the Luhn algorithm, but filter following the Visa pattern.

Template Parameters:
number_of_virtual_value_skippedHelper functions to provide same behavior on a sequence with and without check digits. No "real" value in the sequence will be skipped.

Verify that a number matches the Visa pattern.

Parameters:
current_valid_valueis the current valid value analysed.
current_value_positionis the number of valid value already counted (the current value is not included).
This is also the position (above the valid values) of the current value analysed (0 <= valid_value_counter < n).
Exceptions:
std::invalid_argumentif the first character is not equal to 4. The exception contains a descriptive message of what was expected.
Remarks:
This function use the macro VISA_SIZE to find the real position from left to right.
