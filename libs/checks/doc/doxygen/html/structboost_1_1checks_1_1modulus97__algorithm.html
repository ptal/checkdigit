Detailed Description

template<typename mod97_weight, typename iteration_sense, unsigned int number_of_virtual_value_skipped = 0>
class boost::checks::modulus97_algorithm< mod97_weight, iteration_sense, number_of_virtual_value_skipped >

This class can be used to compute or validate a checksum with a basic modulus 97.

The mod97-10 algorithm (ISO/IEC 7064:2003 Information technology -- Security techniques -- Check character systems) uses two check digits.

Template Parameters:
mod97_weightmust meet the weight concept requirements.
iteration_sensemust meet the iteration_sense concept requirements.
number_of_virtual_value_skippedHelp functions to provide same behavior on sequence with and without check digits. No "real" value in the sequence will be skipped.
Remarks:
This algorithm use two check digits.

Compute the two check digits with a simple modulus 97.

Precondition:
checkdigits should have enough reserved place to store the two check digits.
Postcondition:
The two check digits are stored into checkdigits.
Template Parameters:

Parameters:

checkdigitsis the output iterator in which the two check digits will be written.
