Detailed Description

template<typename mod11_weight, typename iteration_sense, unsigned int number_of_virtual_value_skipped = 0>
class boost::checks::modulus11_algorithm< mod11_weight, iteration_sense, number_of_virtual_value_skipped >

This class can be used to compute or validate checksum with a basic modulus 11.

Template Parameters:
mod11_weightmust meet the weight concept requirements.
iteration_sensemust meet the iteration_sense concept requirements.
number_of_virtual_value_skippedHelp functions to provide same behavior on sequence with and without check digits. No "real" value in the sequence will be skipped.
Remarks:
The range of the check digit is [0..10], the tenth element is translated as the letter 'X'.

translate the current value into an integer valid value.

Template Parameters:

Parameters:
current_valueis the current value analysed in the sequence that must be translated.
valid_value_counteris the number of valid value already counted (the current value is not included).
This is also the position (above the valid values) of the current value analysed (0 <= valid_value_counter < n).
Exceptions:
boost::checks::translation_exceptionis thrown if the translation of current_value failed.
The translation will fail if the current value is not a digit (0 <= i < 10), unless it is the rightmost digit, when the value 10 or the 'x' or 'X' character is allowed.
Returns:
the translation of the current value in the range [0..10].
