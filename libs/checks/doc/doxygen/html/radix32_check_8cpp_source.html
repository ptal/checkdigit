00010 // Version of check used by Mastercard, VISA, and most other credit card companies,
00011 // to allow radix (or base) other than 10, for example 32, 36 and 64.
00012 // Version using permutations devised by Joseph A. Gallian,
00013 // University of Minnesota, Duluth, USA.

00021 // The characters representing the digit values may be assigned arbitrarily
00022 // using a lookup table, for example using code similar to
00023 // Gene Callahan, Dr Dobb's Journal, Dec 1995, 131, 132 & 149.
00024 // Generating Sequential keys in an Arbitrary Radix.
00025 // For example, a radix 42 (a Douglas Adams's Memorial Radix) might use the following
00026 // "0123456789abcdefghijklmnopqrstuvwxyz!#$%&~"
00027 // to represent integer values from 0,1,2 ... 40,41  where, for example,
00028 // 0 is represented by '0', 9 by '9', 15 by 'f', ... 40 by '&' and 41 by '~'
00029 // Some letters might be omitted to avoid confusion with digits ('0' and 'O')
00030 // and to reduce the risk of offensive words appearing.
00031 
00032 // Is an even/odd weighted code.
00033 
00034 // Digits in the even positions (numbering from the right) are multiplied by 2,
00035 // All digits are them summed
00036 // and a check digit added to make the result
00037 // evenly divisible by 10 (sum mod 10 is 0) or ((sum % 10) == 0).
