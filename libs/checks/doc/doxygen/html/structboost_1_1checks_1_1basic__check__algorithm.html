Detailed Description

template<typename iteration_sense, unsigned int number_of_virtual_value_skipped = 0>
class boost::checks::basic_check_algorithm< iteration_sense, number_of_virtual_value_skipped >

The main check algorithm class that provides every static function that can be overloaded.
Most of the functions must be re-implemented to have the desired behavior.

Template Parameters:
iteration_sensemust meet the iteration_sense concept requirements.
number_of_virtual_value_skippedHelper functions to provide the same behavior on sequence with and without checkdigits. No "real" value in the sequence will be skipped.
